# File Sorter by Extension

A Python script that organizes files into folders based on their extensions.

## Description
This project scans a folder and moves files into subfolders named after their file extensions.

## Features
- Automatic folder creation
- File organization
- Works for all file types
- Simple and useful

## Technologies Used
- Python
- os module
- shutil module

## How to Run
1. Clone the repository
2. Navigate to the project folder
3. Run:

   python file_sorter.py

4. Enter a folder path

## Example

Before:
file1.txt, image.jpg, script.py

After:
txt/file1.txt  
jpg/image.jpg  
py/script.py
